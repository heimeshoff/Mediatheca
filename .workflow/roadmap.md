# Roadmap — v1 Finish Line

### M1: In Focus
**Status:** Not Started
**Target:** Cross-cutting "In Focus" concept across all media types

**Goals:**
- Movies get an In Focus toggle flag (auto-clears on watch session)
- TV Series get an In Focus toggle flag (auto-clears on first episode watched)
- Games get InFocus as a new status in the lifecycle (Backlog → InFocus → Playing → ...)

**Tasks:** 001, 002, 003, 004, 005

### M2: Unified Dashboard
**Status:** Not Started
**Target:** Tabbed dashboard (All/Movies/TV Series/Games) replacing the current basic dashboard

**Goals:**
- All tab: TV Series Next Up, Movies In Focus, Games In Focus, Games Recently Played
- Individual media tabs with stats and details, expandable over time
- Intent-driven landing page answering "what's next?"

**Tasks:** 006, 007, 008, 009, 010

### M3: Steam Description Backfill
**Status:** Not Started
**Target:** Backfill missing descriptions for existing games during Steam import

**Goals:**
- During Steam library import, detect games with missing descriptions and fetch from Steam Store API

**Tasks:** 011

### M4: HowLongToBeat Integration
**Status:** Not Started
**Target:** Fetch and display average game completion times from HowLongToBeat

**Goals:**
- F# HTTP client for HLTB's internal API (no official API exists)
- Completion time comparison on game detail page
- Integration into Games dashboard tab

**Tasks:** 012, 013
